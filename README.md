# 🛰️ Broadcast-Groupware

방송국 내부 업무 효율화를 위한 그룹웨어 시스템
캘린더, 예약, 근태, 게시판, 전자결재 등 방송국 업무를 통합 관리합니다.



# 🧩 프로젝트 개요

## 개발 목적

방송국 내부 업무를 웹으로 통합 관리하여 효율화

## 주요 특징

캘린더, 예약, 근태, 게시판, 전자결재 등 필수 업무 통합

역할 기반 기능 분리 (관리자 / 직원)

일정 알림, 예약 중복 방지, 결재 진행 상태 확인 등 업무 편의 기능 제공




# 👥 사용자 역할 및 기능

## 📌 관리자

 회원 관리 (직원, 결재자)

 회의실, 편집실, 차량 예약 관리

 캘린더 일정 등록/수정/삭제

 게시판 관리 및 공지사항 등록

 전자결재 확인 및 승인/반려 처리

## 👨‍💼 직원

 개인 일정 확인 및 관리

 예약 신청 (회의실, 편집실, 차량)

 근태 등록 (출근, 퇴근, 외근)

 게시판 글 작성/수정/삭제

 결재 문서 확인 및 서명 제출
